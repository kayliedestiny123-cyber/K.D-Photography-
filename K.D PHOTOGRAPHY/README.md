# Kaylie Destiny Photography Website

Static website for Kaylie Destiny Photography.

## Pages

- `index.html` - home page, services, testimonial, and inquiry form
- `about.html` - about page and process
- `gallery.html` - filterable gallery
- `pricing.html` - packages, add-ons, and FAQ
- `styles.css` - shared design system and responsive layout
- `script.js` - mobile navigation, sticky header, and gallery filtering

## Quick Customization

- Replace the placeholder image URLs in the HTML files with your own photo file paths or hosted portfolio images.
- Contact email is set to `kayliedestiny123@gmail.com`.
- Instagram footer link is set to `@kayliedestinyphotography`.
- Adjust package names, prices, and deliverables in `pricing.html`.
- The inquiry form currently opens an email draft through `mailto:`. For a production site, connect it to a form service or your hosting provider's form handler.

## Preview

Open `index.html` in a browser, or run a local server from this folder and visit the shown localhost URL.

- Website domain is set to `kayliedestinyphotography.com`.
